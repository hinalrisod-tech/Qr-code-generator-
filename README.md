# Qr-code-generator-
A simple QR code Generator built using python that converts text or URLs into qr code.
##Features
-Generator QR code from text or URL
-Save QR code as image
- Easy to use

- ## Technology Used
- python
- qrcode libarary

- ##How to Run
- install required library.
- pip install qrcode(pil)

 -run the program.
 -python main.py

 ## Use Case
 you can generate Qr codea for links,contact       info, or any text data
